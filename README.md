# Twitter clone

## Environment file

```
//.env
MONGO_URI="<MongoDB connection string>"

TEST_URI="<MongoDB test db connection string>"

JWT_SECRET="<JWT secret key>"

VITE_FIREBASE_API_KEY="<Firebase api key>"
```

## Initialize

```
cd backend
npm i

cd frontend
npm i
```

## Run

```
//Frontend
npm run dev

//Backend
npm run dev
```
